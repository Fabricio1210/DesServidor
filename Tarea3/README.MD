# Tarea 3  
Fue la práctica donde se desarrollaron pruebas unitarias e integrales utilizando `Jest` y `Supertest`.  
El objetivo fue validar el funcionamiento correcto de las operaciones básicas (sumar, restar, multiplicar, dividir) y los endpoints creados con Express, incluyendo la autenticación mediante JWT.

Se realizaron pruebas que comprueban el comportamiento del middleware de autenticación, los endpoints /test y /admin.

---

## Cómo correr el proyecto
De esta forma podemos ejecutar el proyecto y las pruebas:

### Para iniciar el servidor:
```bash
npm start
```

## Tecnologías Usadas

- **Node.js**
- **Express**
- **Jest**
- **Supertest**
- **JSON Web Token (JWT)**
- **dotenv**
