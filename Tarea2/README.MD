# Tarea 2
Fue la practica donde se desarrollo el tema de los sockets usando `socket.io` se logro construir una pequeña aplicacion de caht donde se tienen 3 salas que se pueden cambiar entre si, la unica condicion es que se tenga un nombre para poder identificar al usuario

## Como correr el proyecto
De esta forma podemos correr el proyecto:
```
npm run dev
```

##  Tecnologías Usadas

* **Node.js**
* **Express**
* **Socket.IO** 
* **TypeScript**
* **Handlebars (HBS)**
* **JavaScript (JS)**
* **HTML / CSS**
